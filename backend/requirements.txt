# ============================================
# SafeHer Backend — Slim Python Dependencies
# ============================================
# Install: pip install -r requirements.txt
#
# This is the SLIM install used by default — it boots on Render's
# 512MB free tier without OOM. The /chat endpoint uses the direct
# Gemini/Groq path (RAG_DISABLED=true) which doesn't need any of
# the heavy ML packages below.
#
# For the full retrieval-augmented chat (Standard plan / 2GB+):
#   pip install -r requirements.txt -r requirements-rag.txt
#   RAG_DISABLED=false LITE_MODE=false
#
# IMPORTANT for Render deployment:
#   - runtime.txt pins Python to 3.12.x
#   - Do NOT pin numpy/pandas/scikit-learn to exact versions — let pip
#     pull the latest version that has prebuilt wheels for Python 3.12.
#     Otherwise pip falls back to compiling from source (10-30 min build).
# ============================================
